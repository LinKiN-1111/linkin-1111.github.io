FileStream工具使用方法

命令使用

create 创建文件流
enum 列举文件流
delete 删除文件流
write 写入内容到文件流
append 增加文件至文件流
launch 执行文件流内容
dump 读取文件流内容

文件流（即寄生文件）后缀默认即可

不用命令时使用

命令行创建文件流：
写入文件流：echo 所需内容 >>文件名.格式名(宿主）:文件名.格式名（寄生文件） 理论上可以任意格式，按所需去使用。
寄生文件寄生在宿主之中，宿主被删除，寄生文件被删除，即使重创一个同名文件，也没有寄生文件，除非按照以上格式再写一次。

使用脚本创建文件流

命令使用 (不会用时可不加参数，直接查看使用方法。)
FileStreams.exe enum 宿主 查看宿主内的所有文件流

FileStreams.exe create (当前路径或自定义绝对路径）宿主 寄生文件名 <寄生文件格式，可默认>

FileStreams.exe write 宿主 寄生文件名 <格式名，对应好> content 均不得空格
寄生文件的内容

FileStreams.exe dump 宿主 寄生文件名 <格式名，对应好> 查看的字符数

FileStreams.exe append 宿主 寄生文件名 绝对路径+所需启动文件

FileStreams.exe launch 宿主 寄生文件 执行文件流的内容
