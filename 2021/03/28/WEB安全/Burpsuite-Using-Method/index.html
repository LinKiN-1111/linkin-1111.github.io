万能的Burpsuite

了解proxy（代理）

burpsuite作用

1.拦截代理（Proxy），检查和更改了浏览器与目标应用程序间的流量；
2.可感知应用程序的网络爬虫（Spider），完整地枚举应用程序的内容和功能；
3.高级扫描器，执行后自动发现web应用程序漏洞
4.入侵测试工具（Intruder），执行强大的定制攻击去发现和利用不同寻常的漏洞；
5.重放工具（Repeater），靠手段操作来触发单独的HTTP请求，分析应用程序响应
6.会话工具（Sequencer），用于分析不可预知的应用程序会话令牌和重要数据项的随机性工具；
7.解码器，进行手动执行或对应用程序数据者智能解码编码的工具；
8.扩展性强，可以加载使用第三方扩展或者自己的扩展。
Key：
1.运行依赖java环境，需配置java环境
2.设置JVM内存大小（默认为64M），过小处理大规模任务会出现问题。这里可以设置它预算内存大小。单位可以M或G。 java -jar -Xmx2G BurpLoader.jar 即在-jar之后设置预设内存，在启动快捷键中设置即可，若CMD启动则要自行设置。
