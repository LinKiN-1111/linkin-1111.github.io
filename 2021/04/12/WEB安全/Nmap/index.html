Nmap快速上手

Nmap是什么？

Nmap用于在远程机器上探测网络，执行安全扫描，网络审计和搜寻开放端口。他会扫描远程在线主机，该主机的操作系统，包过滤器和开放的端口。
有命令行界面和可视化界面，可根据自己的喜好进行使用。

How to Use？

  1. HOST Scan：nmap host_address
  2. IP Scan : nmap ip_address
  3. Know More: -v
  4. Scan More: nmap ip1 ip2 ip3 /nmap host1 host2 host3
  5. Scan range: nmap 192.168.0./192.168.0.1,2,3（扫描0 1 2 3）/192.168.0.0-100（扫描0-100） 扫描C段，即号为0-255
  6. Scan Target From File List: nmap -iL target.txt
  7. Exclude Someone: nmap 192.168.0.* –exclude 192.168.0.1 (不包含）
