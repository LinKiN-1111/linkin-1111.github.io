linux学习笔记

终端格式

打开终端后的提示:（以kali系统为例）

  1. (root@kali)-[~]$
  2. (root@kali)-[/home/linkin]#

[当前登录用户名@主机名称]-[当前所在路径]#/$
#表示身份为超级管理员
$表示身份普通用户

首先了解当前系统信息

硬盘大小

所用命令:fdisk -l
/dev/sda 操作系统中第一块硬盘的名称以及所在的路径
linux操作系统中一切皆文件（即都有文件名） sd(硬盘类型) a(第一块)

内存大小

cat /proc/meminfo
注意total即可。。。

cpu大小

cat /proc/cpuinfo

关机和重启的命令

关机:poweroff
重启:reboot

linux操作系统

linux操作系统树状结构

/ — 操作系统的起始路径(根路径)
目录 == 文件夹
..(上级目录)
cd (change direction)
pwd (显示当前所在目录路径)
ls (list，列出当前目录内容）

根目录下的目录作用

  1. /bin — 普通用户和管理员都可以执行的命令字(例如CAT)
  2. /sbin — 只有管理员才能执行的命令 (例如开机重启)
  3. /boot — 引导 主引导目录 独立分区 启动菜单 内核(vmliuz)
  4. /dev — device设备 设备文件存放目录
  5. /etc — 配置文件存放目录
  6. /home — 普通用户的家目录
  7. /root —管理员的家目录
  8. /media — 光驱的挂载目录
  9. /mnt — 临时设备挂载目录
  10. /proc —里面的数据都在内存中，进程的所在目录
  11. /tmp —临时文件存放目录
  12. /usr —软件的安装目录
  13. /var —常变文件存放目录（日志文件、邮件文件）
  14. 后面的暂时不重要，先不要求记忆
  15. /lost+found — 暂时不重要，没讲
  16. /lib(lib64) —例如可以用于存放python包（即函数库）之类的东西（暂时不重要）

快捷键应用

安装VMwaretools

该工具可以实现虚拟机和真实机之间的文件复制 （txt文件等编码要改为UTF-8）因为windows默认编码是ANSI；linux是UTF-8.需要编码对应

快捷键功能

  1. Tab键功能 补齐命令字和已存在文件名称补齐
  2. 清楚屏幕内容 ctrl+l
  3. 终止快捷键 ctrl+c

linux系统中如何分辨文件类型

蓝色（kali下蓝色） — 目录（文件夹）
黑色（kali下白色）— 普通文件（大部分可以cat）
浅蓝色（kali下浅蓝色） — 符号链接（即windows下的快捷方式）
黑底黄字（kali下橘色） — 设备文件 硬盘 sda
绿色（kali下绿色） — 带有执行权限的文件
红色（kali下红色） — 压缩文件
紫色：（kali下紫色） — 图片 模块
kali下：
红色闪烁：表示链接的文件有问题
灰色：表示其它文件

基础命令（增删改查）

查询命令

查询文件：查询目录下有什么内容、查看文件中的内容

查询目录下有什么内容

ls

查看文件中的内容

cat

创建命令

创建命令：创建文件、创建目录

创建文件

touch 文件名 echo “写入内容” > (绝对路径)文件（没有会创建）

创建目录

mkdir 目录名 make directory

改动命令

改：剪切、复制

剪切

mv mv 文件 目录
同时mv有重命名的作用 mv 文件名 文件名

复制

cp cp 文件名 目录（在文件目录下复制一个文件）
cp 文件1 文件2 （创建一个文件2内容与文件1）

符号链接（快捷方式）

创建符号链接ln -s ln -s 源文件（带绝对路径） 建立的链接文件（绝对路径）
最后在绝对路径下可带名字也可不带（是否重命名而已）

删除命令

rm remove移除 rm 文件名 或 rm -f 文件（强制删除）
rm -rf 目录(要用-rf才能删除 因为目录下有多个文件，需要循环删除)
