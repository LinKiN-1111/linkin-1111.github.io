linux_Learning_Note4

查看基本信息

查看CPU信息

cat /proc/cpuinfo

查看内容信息

cat /proc/meminfo

查看硬件信息

lspci 例：lscpu，也可以查看CPU信息。。。

硬盘使用情况

df -TH

查看系统相关信息

我是谁：whoami
还有谁：who
Linux内核版本信息：uname -a
Linux发发行版本信息:1.redhat系列：cat /etc/redhat-release ;2.debian系列:cat /etc/issue

查看进程

top：相当于windows里面的任务管理器（q退出）
ps 参数：-a列出当前活跃进程，-e列出所有进程，-ef列出更详细的信息，-aux也是列出更详细的信息

中断、停止进程

中断进程（即后台运行）

Ctrl+C为中断进程，并未结束进程。

停止进程

Ctrl+z为停止进程，进程停止。
找到进程：ps -e|grep 进程名字
中止进程：kill 参数 PID即可结束进程。。。参数：-15正常结束；-9强制结束

查看运行服务

chkconfig --list：查看所有服务的启动状态
chkconfig --list|grep 服务名
上方命令为旧版系统使用，不建议使用。。。
systemctl status 服务名
systemctl start|restart|stop 服务名：开启/关闭服务

系统网络状态

查看ip地址：ip a(address) show
网络连通性：ping url
有哪些进程联网：ss -atunp：tcp.udp,n:有效;p:进程

软件管理

yum命令（在kali系统下需要安装才可使用）
yum update ：更新镜像源（不会更新软件）
yum search ：没用过，应该是查找软件。。。
yum install 软件名：安装软件 例：yum install vim
yum remove 软件名：删除软件 例：yum remove vim
其中kali为debian系，所以要使用apt。。。

SSH远程管理

Linux远程管理软件:putty、Xshell、SecureCRT

Linux帮助命令

帮助–help或-h
man 命令其中PageDown下翻、PageUP上翻、home开头、End结尾、/string搜索、？String、q离开
Info也可以。。。
