linux_Learning_Note2

特殊目录

. —当前目录
.. —上一级目录
- —上一个目录
~ —家目录(普通用户home，管理员用户root)

常用命令

新建目录

mkdir 目录
参数: -p —可以直接创建多级目录

删除目录

rmdir 目录 删除空目录
参数:-p 可以删除有文件的目录（经过尝试实际上删不了）
建议使用rm -r进行有文件目录的删除
小心rm -rf 该命令十分危险。。。
rm -i 交互删除，暂时不知道有什么用。。。

查看

ls
参数:-a —可以查看所有文件件
-l —以长模式列出
-d —仅列出目录内容（好像只会列出.）
-h —以易读的方式列出（无法单独使用）

cat或more|less
cat —查看文件全部内容
more |less—一页一页查看文件内容420

复制剪切

cp
参数:-a—连同属性和权限一起复制
-r—递归复制（应该和rm那个差不多）
-f—force强制
mv和cp差不多。。。
