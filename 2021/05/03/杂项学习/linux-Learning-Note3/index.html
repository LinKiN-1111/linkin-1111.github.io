档案权限

linux三种权限 user-用户;group-用户组;other其他用户…
在linux中，任何一个档案都具有所有者、用户组、其他用户这三种身份的个别权限
ls -l的代表内容:

权限 连接数 拥有者 用户组 大小 修改日期 档案名
drwxr-xr-x 3 linkin linkin 4.0K 5月 3 17:27 123

其中权限每个字母代表:
以d|rwx|r-x|r-x为例:
第一个字符代表档案的类型：
d代表是目录;
-代表是文件;
l代表是链接文件;
b代表是设备;
c代表是字符设备（如键盘、鼠标）
后面分别代表用户、用户组、其它用户的三个权限
r代表读；w代表写；x代表执行

改变权限

chown(change owner) 改变档案所属用户（只能在root条件下使用）
chown 用户 文件
ps：若没有用户，可用useradd 用户 来添加用户
参数：-R用于修改目录（递归，即把全部文件都改成用户）

chgrp(change group) 改变档案所属用户组 （用法和chown一致）

chmod 更改权限，即把用户，用户组，其它用户的权限进行设置
chmod xyz (-R) 文件或目录
xyz分别代表三个数字
r:4 w:2 x:1
举例说明:
chmod 567 (-R) 文件或目录
即说明文件权限改为了r-x|rw-|rwx 以此类推。

还有一种用法：
chmod ugoa +/-/= rwx 文件或目录
u表示用户 g表示用户组 o表示其它用户 a表示所有

快捷键应用

  1. Tab：命令补齐以及文件补齐
  2. Ctrl+c：中断操作
  3. Ctrl+d：键盘输入结束，相当于exit
  4. Ctrl+a：跳到开头
  5. Ctrl+e：跳到结尾
  6. Ctrl+u：删除光标前
  7. Ctrl+k：删除光标后

文件编辑（vim）

vi 文件
一开始为一般模式
i o a进入编辑模式
一般模式下直接输入:为指令模式
vim自行另外学习吧。。。
